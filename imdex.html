Типы данных
Примитивные Свойство Сложные Свойство
Underfined undefined и указывает, что значение не установлено Object Object: представляет комплексный объект
Boolean Boolean: представляет логическое значение true или false
Number Number: представляет числовое значение
String String: представляет строку
BigInt BigInt: предназначен для представления очень больших целых чисел
Symbolк Symbol: представляет уникальное значение, которое часто применяется для обращения к свойствам сложных объектов
Null Null: представляет одно специальное значение - null и указывает на отсутствие значения
